# Binary Search Number Guessing Game

## About
This project demonstrates how a computer tries to guess YOUR number using binary search!

## How to Play
1. Think of a number between 1 and 100
2. The computer will make a guess
3. Tell it if the guess is 'high', 'low', or 'correct'
4. Can it guess it 5 tries?

## Key Concepts 
- Binary Search
- Algorithms
- Conditional Logic
